SQL Basics

SQL (Structured Query Language) is used for managing and manipulating relational databases. Common operations include querying, updating, and maintaining data in relational databases like MySQL, PostgreSQL, and SQLite.

Basic Syntax

Example of a simple SQL query to fetch customers with age greater than 25: 

SELECT name 
FROM customers 
WHERE age > 25;

Intermediate SQL

Intermediate SQL introduces JOINs, subqueries, and grouping data using aggregate functions like COUNT, AVG, and SUM. These concepts are essential for analyzing and combining complex datasets.

JOIN Example

SELECT orders.id, customers.name 
FROM orders 
INNER JOIN customers 
ON orders.customer_id = customers.id;

Advanced SQL

Advanced SQL topics include stored procedures, triggers, transactions, and advanced query techniques like window functions and CTEs. These are vital for building robust database systems.

Window Function Example

SELECT name, salary, 
RANK() OVER (ORDER BY salary DESC) AS rank 
FROM employees;
