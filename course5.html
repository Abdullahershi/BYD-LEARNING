Introduction to HTML

HTML (HyperText Markup Language) is the foundation of the web, defining the structure and layout of a webpage. It works hand-in-hand with CSS (Cascading Style Sheets) to create visually appealing websites.

In this guide, you'll learn the following:

  • How to structure an HTML document
  • Common HTML elements and attributes
  • Styling using CSS
  • Interactive features and semantic HTML

HTML Structure

An HTML document follows a specific structure. Here's a detailed breakdown:

Tag Purpose
<!DOCTYPE html> Specifies the document type and version of HTML (e.g., HTML5).
<html> Wraps the entire HTML document content.
<head> Contains metadata, links, and scripts for the page.
<body> Holds the visible content of the page.

HTML Attributes

Attributes provide additional information about an element. Common attributes include:

  • id: Assigns a unique identifier to an element.
  • class: Groups elements for styling or scripting.
  • src: Specifies the source for images or scripts.
  • alt: Provides alternative text for images.
  • href: Links to another webpage or resource.

HTML Elements

HTML includes a variety of elements for text, images, and multimedia:

  • Headings: <h1> to <h6> define titles and subtitles.
  • Paragraphs: <p> for blocks of text.
  • Links: <a> for navigation.
  • Lists: Ordered (<ol>) and unordered (<ul>).
  • Images: <img> to embed pictures.

CSS Basics

CSS is used to style HTML. Here are key concepts:

  • Selectors: Target elements (e.g., h1, .class, #id).
  • Properties: Define styles like color, font-size, and margin.
  • Box Model: Includes margins, borders, padding, and the content itself.
