Introduction to Python

Python is a high-level, interpreted programming language that is widely used for web development, data analysis, machine learning, automation, and more.

  • Simple and easy-to-read syntax
  • Great for beginners and professionals
  • Extensive libraries and frameworks

Python Syntax

Python uses indentation to define code blocks, unlike many other programming languages that use braces. 

def greet(name):
    print(f"Hello, {name}!")

greet("Python Learner")

Variables and Data Types

In Python, variables are dynamically typed, meaning you don’t need to explicitly declare the type of a variable.

  • String: 'Hello'
  • Integer: 10
  • Float: 10.5
  • List: [1, 2, 3]
  • Boolean: True / False

Control Structures

Python provides several control structures like loops and conditionals for decision-making:

Control Structure Description Example
if-else Executes code based on a condition if x > 10: print("Greater")
for loop Iterates over a sequence for i in range(5): print(i)
while loop Executes code while a condition is true while x > 0: print(x); x -= 1

Functions in Python

Functions allow you to reuse code and organize your program into smaller, manageable pieces. In Python, you define a function using the `def` keyword. 

def add(a, b):
    return a + b

result = add(3, 4)
print(result)  # Output: 7

Example Python Program

Here’s a simple program that calculates the factorial of a number using recursion: 

def factorial(n):
    if n == 0:
        return 1
    else:
        return n * factorial(n - 1)

print(factorial(5))  # Output: 120
