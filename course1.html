Introduction to SQL

SQL (Structured Query Language) is the standard language for managing relational databases. With SQL, you can:

  • Retrieve and manipulate data
  • Create and maintain database structures
  • Ensure data consistency and security
-- Example: Fetching data from a table
SELECT name, age FROM employees WHERE age > 30;

CRUD Operations

CRUD stands for Create, Read, Update, and Delete, the fundamental operations of a database:

Operation SQL Statement
Create INSERT INTO table_name (columns) VALUES (values);
Read SELECT columns FROM table_name WHERE condition;
Update UPDATE table_name SET column=value WHERE condition;
Delete DELETE FROM table_name WHERE condition;
-- Example: Updating a record
UPDATE employees SET age = 35 WHERE name = 'John Doe';

Joins

Joins allow combining data from multiple tables based on relationships. Common types of joins include:

  • Inner Join: Returns matching rows from both tables.
  • Left Join: Returns all rows from the left table and matching rows from the right.
  • Full Join: Combines all rows from both tables.
-- Example: Inner Join
SELECT orders.id, customers.name
FROM orders
INNER JOIN customers ON orders.customer_id = customers.id;

Views

Views simplify complex queries and provide a virtual table based on the result set of a SQL query.

-- Example: Creating a view
CREATE VIEW high_earners AS
SELECT name, salary FROM employees WHERE salary > 100000;

Advanced SQL Concepts

Advanced techniques include stored procedures, triggers, and performance tuning:

  • Stored Procedures: Predefined SQL queries for repeated use.
  • Triggers: Automatic actions triggered by database events.
  • Indexing: Enhances data retrieval speed.
-- Example: Creating a trigger
CREATE TRIGGER after_insert_employee
AFTER INSERT ON employees
FOR EACH ROW
BEGIN
INSERT INTO logs (message) VALUES ('New employee added');
END;
