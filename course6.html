Introduction to C++

C++ is a powerful, high-performance programming language used in software development, game development, and systems programming.

  • Object-oriented programming
  • Supports low-level and high-level programming
  • Widely used in competitive programming

C++ Syntax

The basic structure of a C++ program is:

#include <iostream>
using namespace std;

int main() {
    cout << "Hello, World!" << endl;
    return 0;
}

Data Types in C++

Data Type Description Example
int Stores integers 10, 20, -100
float Stores decimal numbers 3.14, -2.5
char Stores single characters 'A', 'z'
string Stores text "Hello, World!"
bool Stores true or false values true, false

Control Statements

C++ provides various control statements to manage the flow of your program:

  • if/else: Conditional branching
  • for: Iterative loops
  • while: Repeated loops until a condition is met
  • switch: Multiple condition checks

Examples

#include <iostream>
using namespace std;

int main() {
    int a = 5, b = 10;
    cout << "Sum: " << a + b << endl;
    return 0;
}
