Introduction to CSS

CSS (Cascading Style Sheets) is a language used to style the layout, colors, fonts, and overall design of HTML elements.

  • Separates content (HTML) from design
  • Improves web page consistency
  • Supports responsive designs

CSS Syntax

The basic syntax of CSS consists of a selector, property, and value: 

selector {
    property: value;
}

CSS Selectors

CSS uses selectors to target HTML elements:

Selector Target Example
Element Specific HTML element p { color: red; }
Class Elements with a specific class .example { font-size: 16px; }
ID Element with a specific ID #unique { margin: 10px; }

Common CSS Properties

Some of the most commonly used CSS properties include:

  • color: Sets the text color
  • background-color: Sets the background color
  • font-size: Adjusts the font size
  • margin: Adds space outside an element
  • padding: Adds space inside an element

CSS Animations

CSS allows you to create engaging animations. For example: 

@keyframes example {
    from { background-color: red; }
    to { background-color: yellow; }
}

div {
    animation: example 5s infinite;
}
